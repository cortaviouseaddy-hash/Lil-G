Lil-G assistant

Talk to Lil-G and hear it respond.

Type a message, tap the mic, or start wake listening and say "Hey Lil-G" if your browser supports speech recognition. Lil-G replies in chat, can search the internet, and can remember facts on this device.

Install it on your phone

Android can show an install button. On iPhone, open this site in Safari, tap Share, then choose Add to Home Screen.

Memory

Say "remember that..." and Lil-G will save it on this device.

Lil-G assistant

Talk to Lil-G and hear it respond.

Type a message, tap the mic, or start wake listening and say "Hey Lil-G" if your browser supports speech recognition. Lil-G replies in chat, can search the internet, and can remember facts on this device.

Install it on your phone, tablet, laptop, or desktop

Chrome, Edge, and Android can show an install button. On iPhone or iPad, open this site in Safari, tap Share, then choose Add to Home Screen.

Memory

    Voice and device settings
    Voice style

    Choose one of six talk-back styles. The exact system voice depends on your browser.

    Profile sync

    Save a local profile name, export a sync code, and import that code on another installed copy so Lil-G recognizes the same memories, voice settings, and avatar settings there too.

    Quick launch

    Open supported web apps or browser pages in a new tab. Some devices may hand these links off to installed apps.

    Screen context

    Share your screen when you want Lil-G to know you are looking at something. Full screen reading still needs a future OCR or vision service, so paste text when you want exact help.

    Screen sharing is off.

    Native app and Discord access

    Lil-G can listen while this app is open and the browser allows the microphone. Reading Discord directly, sending messages, controlling other apps, showing system pop-ups, or listening after the app is closed requires a native companion app, approved Discord bot or OAuth integration, and operating-system permissions.
